# herokuapp

#### Tech stack: Java 8, Maven and libraries: Selenide, Allure, Testng 

### Set up

You will need the following technologies available to try it out:
* Git
* Maven 3+
* JDK 8
* IDE of your choice 
* Web browser Chrome

### How to run 

```mvn clean test```

### Generate Allure report 

```mvn allure:report```

### How to open in browser

```mvn allure:serve```
